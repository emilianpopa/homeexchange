# HomeExchange Project — Claude Instructions

## Git Workflow (CRITICAL)
Always follow this order:
1. `git pull` from GitHub **before** making any local changes
2. Make changes locally
3. `git commit`
4. `git push` to GitHub **after** every commit

GitHub is the single source of truth. The user works on multiple machines, so local repos may be outdated. Never modify local files without pulling first.
